# NovaSphere - Site com Login

Projeto de site com autenticação funcional em JavaScript puro.

## Páginas
- `index.html`: tela inicial de login/registro com header profissional e footer.
- `home.html`: página principal após login com validação de sessão.

## Como executar
Basta abrir `index.html` no navegador ou usar um servidor local:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.
